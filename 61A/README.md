# A. Ultra-Fast Mathematician

Problem source: https://codeforces.com/problemset/problem/61/A

Time limit per test: 1 second
Memory limit per test: 256 megabytes

## Problem

Given two binary strings of equal length, produce a new binary string where each character is '1' if the corresponding characters of the two input strings differ, and '0' otherwise.

## Input

- Two lines, each containing a binary string of the same length.

## Output

- A single line: the resulting binary string as described above.
